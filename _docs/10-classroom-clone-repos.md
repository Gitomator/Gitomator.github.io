Clone repos to a local directory on your machine.

Usage:

```sh
 $ gitomator-clone-repos REPOS-CONFIG LOCAL-DIR
```

### Minimal configuration

Specify the repo names

```yaml
repos:
  - repo-001
  - repo-002
  - repo-003
  - repo-004
```
